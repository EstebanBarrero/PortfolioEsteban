# Assets Directory

## How to add your images:

### avatar.jpg
- Replace this file with your profile photo
- Recommended: 400x400px square, JPG/PNG
- Used in: Hero section (orbit center) + About section

### projects/
- Add project screenshots here
- Recommended: 800x450px (16:9 ratio), JPG/PNG
- Name them to match `image` field in `src/data/projects.ts`

### cv.pdf
- Place in `public/assets/cv.pdf` (NOT in src/assets)
- Linked from the "Download CV" button in Hero

## Image naming convention:
```
projects/ai-chat.jpg
projects/serverless.jpg
projects/ecommerce.jpg
projects/dashboard.jpg
projects/rag.jpg
projects/mcp.jpg
projects/code-review.jpg
projects/microservices.jpg
projects/banking-api.jpg
```
